# Les RH O’naturel — WordPress Showcase Website

Showcase website developed with WordPress to present human resources and nature-based wellbeing support services.  
Live website: https://www.lesrhonaturel.fr/

## Project Goal

Create a clear, calming, and professional online presence for a human-centered activity focused on nature, balance, and personal development.

The website needed to:
- Inspire trust  
- Highlight the human dimension of the services  
- Remain easy for the client to update  

## Technical Stack

- WordPress  
- Theme based on the "Variations" theme (lightweight and adaptable structure)  
- Customizations via CSS and template adjustments  
- Creation of a tree-style navigation menu using an HTML block  
- Responsive optimization (mobile / tablet)  
- Basic performance optimization  

## Work Performed

- Visual adaptation of the theme to match the brand’s natural and human-centered identity  
- Structuring of service and support pages  
- Implementation of a clear content hierarchy  
- Style customizations (typography, colors, spacing)  
- Optimization of readability and overall user experience  

## Technical Approach

Customizations were implemented following WordPress best practices:
- Clear separation between the parent theme and specific adjustments  
- Non-intrusive code additions to ensure maintainability  
- Structure designed to allow the client to manage content easily  

## Status

Project currently in production.  
The code in this repository represents the customizations developed for the website.
